# OSCER

## Overview

OSCER (Open Source Contribution and Engagement Resources) is Nava's initiative to strengthen open source practices and community engagement in the civic technology space. It provides resources, tooling, and frameworks that help organizations — from government agencies to civic tech nonprofits — participate effectively in the open source ecosystem.

## Why It Matters

Building effective government services in the open requires more than good code — it requires clear practices around licensing, contribution, community management, and collaboration. OSCER builds capacity across the civic tech ecosystem by making it easier for organizations to adopt open source practices that serve the public interest. When more organizations can participate effectively in open source, the tools and services that reach people get better for everyone.
