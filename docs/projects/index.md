# Our Open Source Projects

We build open source tools and platforms that help government agencies deliver simple, effective services to the people who need them. Our projects range from platforms that directly serve beneficiaries to shared infrastructure and resources that build capacity across the civic tech ecosystem.

Below are Nava's featured open source projects. Each page covers what the project does, why it matters, and how to get involved.

## Featured Projects

### [Strata](strata.md)
A platform for building and delivering government benefit services.

### [OSCER](oscer.md)
Open Source Contribution and Engagement Resources for civic technology.

---

## Getting Involved

Every project has its own contribution guidelines and community. Visit the individual project pages to learn how to participate, or browse the [navapbc GitHub organization](https://github.com/navapbc) to explore all of Nava's public repositories.

If you're new to Nava's open source work and aren't sure where to start, look for issues labeled `good first issue` across our projects.
