# Strata

## Overview

Strata is Nava's platform for building and delivering government benefit services. It provides shared infrastructure and patterns that help agencies modernize how people access benefits — reducing wait times, simplifying applications, and making services more resilient for the millions of households that depend on them.

## Why It Matters

Every government benefit program faces similar technical challenges: identity verification, eligibility determination, document processing, and case management. Strata provides human-centered building blocks for these common patterns so that agencies can focus on what makes their program unique rather than rebuilding foundational infrastructure from scratch. When we improve a shared component, every program built on Strata benefits.
